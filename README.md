# AI-Based Legacy Code Analyzer & Migrator

This project analyzes large legacy codebases (starting with ERPNext)
to extract:
- Entity schemas
- Business rules
- Relationships
- Migration-ready intermediate representations

## Tech Stack
- Python 3.9+
- AST Parsing
- ERPNext (Frappe Framework)

## Current Features
- Static code analysis
- DocType schema extraction
- Business logic detection
- Relationship mapping
